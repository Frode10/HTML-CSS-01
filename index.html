Mål:

Se hvor mye jeg kan sette opp uten å bruke oppslag/internett. Gikk bra tilogmed enkel flexbox og css grid. Ønsket at første "div" skulle span 2 columns. Fungerer men ikke helt 100% kode.

Bli bedre på:

Må trene mer på css grid og flexbox. (Tenker css grid for layout og flexbox for posisjonering av elementer i grid.) Må også trene mer på DRY. Sjekk ut bedre DRY for tekst og tekstcolor.

Trix:

VScode - ctrl + D velger "flere classer med samme navn" Hold inne ALT og flytt hele linjen. Comment flere linjer Ctrl + H U Ikke bruk class på alt - tenk dry, tenk kanskje typ section div > * {} CSS viktig å huske: Css leser nedover så det siste som er definert vil fungere. Se box og flex-box (bildebox), flytt den over box og fungerer ikke.
