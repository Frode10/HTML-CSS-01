Koding og !design in one go!

Denne siden er responsive for desktop, Ipad og mobil. Testet i Responsive Design Mode i browser.

CSS testet w3school validator med resultat "No Error Found. This document validates as CSS level 3 + SVG" (hva nå enn det betyr..)

HTML testet w3school validator med resultat noen info og warning.

Mål:

Se hvor mye jeg kan sette opp uten å bruke oppslag/internett. Gikk bra tilogmed enkel flexbox og css grid. Ønsket at første "div" skulle span fullwidht. Fikk til noe i css grid. Ikke i flexbox.

Bli bedre på:

Må trene mer på css grid og flexbox. (Tenker css grid for layout og flexbox for posisjonering av elementer i grid.) Må også trene mer på DRY.

Trix:

VScode - ctrl + D velger "flere classer med samme navn" Hold inne ALT og flytt hele linjen. Comment flere linjer Ctrl + H U
